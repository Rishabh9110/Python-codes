Welcome to the **Python Learning Hub**! This repository is a comprehensive collection of Python code, designed to help you master Python programming from the very basics to advanced functional concepts. It serves as both a learning resource and a reference point for daily Python practices.

## 📁 Repository Structure

The repository has been neatly organized into chapters, each focusing on a specific core concept of Python. Every chapter is further divided into basic examples and dedicated `practice_set` folders containing coding problems and solutions.

- **`Chapter_01_Basics/`**  
  Introduction to Python, basic inputs, and simple scripts.
  
- **`Chapter_02_Variables_and_DataTypes/`**  
  Deep dive into Python data types, variables, typecasting, and operators.
  
- **`Chapter_03_Strings/`**  
  String manipulation, string functions (like `len()`), escape sequences, and slicing.
  
- **`Chapter_04_Lists_and_Tuples/`**  
  Understanding collections in Python, indexing, and list/tuple methods.
  
- **`Chapter_05_Dictionary_and_Sets/`**  
  Working with key-value pairs (Dictionaries) and unique, unordered collections (Sets).
  
- **`Chapter_06_Conditional_Expressions/`**  
  Mastering `if`, `elif`, and `else` statements for control flow.
  
- **`Chapter_07_Loops/`**  
  Iterating with `for` and `while` loops, break/continue statements, and generating star patterns.
  
- **`Chapter_08_Functions/`**  
  Writing modular code using functions, default arguments, and exploring recursion.

## ✨ Features

- **Clean Codebase**: All Python files are formatted using [`black`](https://github.com/psf/black) to ensure PEP-8 compliance and readability.
- **Consistent Structure**: Logical separation of concepts into chapters, making it easy to track learning progress.
- **Practice Sets**: Each chapter includes real coding problems to reinforce the concepts taught.

## 🚀 How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rishabh9110/Python-codes.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd Python-codes
   ```
3. **Run any script:**
   Navigate into any chapter and execute a script using Python:
   ```bash
   python Chapter_01_Basics/01_first.py
   ```
